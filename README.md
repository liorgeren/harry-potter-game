# ⚡ Harry Potter Trivia Challenge

A mobile-friendly, single-player trivia game built with React where **you race against an AI opponent** across 15 rounds of Harry Potter questions.

**Made by Chloe Gerenstein**

🌐 **[Play it here](https://liorgeren.github.io/harry-potter-game/)**

---

## How to Play

- **15 rounds**, each featuring a different Harry Potter character as your AI opponent
- **5 questions per round**, ordered from easiest to hardest
- Type your answer and hit Submit — **whoever answers correctly first wins the question**
- The AI will start typing its answer after a delay — race to beat it!
- Win the round (most correct answers) to earn **100 💎 diamonds**; lose and you still get **20 💎**
- Use **10 💎** to buy a hint on any question — the answer is revealed and auto-submitted for you

## Rules

- Answers are **case-insensitive** and allow minor typos
- You have **30 seconds** per question
- Starting diamonds: **50**
- The AI answers correctly ~70% of the time and responds 12–25 seconds after the question appears

- **React 18** + **Vite 5**
- Plain CSS with a dark magical theme (Cinzel font, gold & purple palette)
- Frontend only — no backend, no dependencies beyond React
